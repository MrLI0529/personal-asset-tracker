# 个人资产表

一个简洁实用的个人财务管理工具，帮助你轻松管理资产、负债、收入和支出。

## 功能特点

- 📊 资产、负债、收入、支出全面管理
- 📈 实时计算净资产和月收支差额
- 🎨 饼图可视化展示各项分布
- 🤖 智能财务状况评价系统
- 💾 数据备份与恢复（支持JSON/CSV/自定义格式）
- 💻 纯前端实现，无需服务器

## 使用方法

1. 直接打开 `index.html` 文件
2. 添加你的资产、负债、收入、支出信息
3. 系统自动计算并生成财务评价
4. 可随时导出数据备份

## 数据存储

数据保存在浏览器本地存储（localStorage），不会上传到任何服务器，保护你的隐私。

## 技术栈

- HTML5
- CSS3
- JavaScript
- Chart.js（图表库）
